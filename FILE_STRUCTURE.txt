🎓 KEY CONCEPTS
════════════════════════════════════════════════════════════════════

Hybrid Architecture = Local + Server
├── User edits data
├── Saved locally (instant ✨)
├── Queued for sync
├── Auto-sync to server (30s)
└── Persisted in Supabase

Offline-First = Works without internet
├── All changes saved locally
├── Queue managed automatically
├── Retries when connection returns
└── No data loss

Real-time Sync = Stays in sync
├── Auto-sync every 30 seconds
├── Manual sync available
├── Event notifications
└── Conflict detection
